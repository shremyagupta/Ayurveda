# Shree Sanjeevni - Atyadhunik Panchkarma Vibhag

A simple, clean, and static website for an Ayurveda Department ("Atyadhunik Panchkarma Vibhag"), built using React, Vite, and Tailwind CSS.

## Features
- **Home Page**: Hero section, philosophy overview, and services preview.
- **Services**: Detailed descriptions of 5 Panchkarma therapies (Vaman, Virechan, Basti, Nasya, Raktmokshan).
- **About Us**: Information about the department and its philosophy.
- **Contact**: Functional-looking contact form and location details.
- **Responsive Design**: Fully mobile-responsive navigation and layout.

## Tech Stack
- React
- Vite
- Tailwind CSS

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/shremyagupta/Ayurveda.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
